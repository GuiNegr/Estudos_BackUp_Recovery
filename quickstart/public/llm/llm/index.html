Explorando o Uso de LLMS

2026/02/09

LLMS São uma vertente da utilização de Machine Learning que utiliza um algoritmo chamado Deep Learning para conseguir efetivar padrões de conhecimento. portanto quando falamos de LLMS estamos conversando sobre modelos de linguagems capazem de traduzir matematica em texto, já que utilizando seus milhoes,bilhoes,trilhoes de paramentros são capazes de criar uma probabilidade sobre qual frase se conecta mais com a anterior para assim formar uma resposta ao usuario. por sua vez essa informação acaba se tornando falsas,incoerentes e reais. apesar da grande evolução ainda é necessario uma validação humana sobre aquilo que lhe foi dito ou escrito. como um texto ou mesmo um codigo gerado por ela.
