Um **servidor de aplicação** é um software que:

> **fornece um ambiente completo para executar aplicações, cuidando de infraestrutura que você não quer reimplementar Mais especificamente, o servidor da aplicação é o componente de tempo de execução principal em todas as configurações e onde é executada uma aplicação. O servidor da aplicação colabora com o servidor da Web para devolver uma resposta dinâmica, personalizada para um pedido de cliente.**
